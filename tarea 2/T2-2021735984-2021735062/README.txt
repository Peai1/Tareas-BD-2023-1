Para poder ejecutar el programa se deben seguir los siguientes pasos:
1) Importar base de datos en phpmyadmin, con nombre Prestigetravels.
2) Copiar la carpeta "tareabd" la cual contiene los archivos php, imagenes y css en la ubicación donde se encuentra
instalado xampp en la carpeta htdocs.
3) Ejecutar XAMPP iniciando MySQL y Apache.
4) Ingresar al sitio web local en "localhost/T2-2021735984-2021735062/php".

Algunos supuestos que tomamos son:

- Para hacer uso de la busqueda avanzada, primero se debe ingresar lo que se desea buscar en la
barra de busqueda, donde despues se pueden agregar los filtros que se piden en el enunciado.
- Se debe iniciar sesión primero que todo para poder acceder a las distintas secciones de la página.

En la tabla "resenas", las calificaciones hacen referencia a los siguientes campos:

En hoteles:
calificacion_1 = Limpieza.
calificacion_2 = Servicio.
calificacion_3 = Decoración.
calificacion_4 = Calidad de las camas.

En paquetes:
calificacion_1 = Calidad de hoteles.
calificacion_2 = Transporte.
calificacion_3 = Servicio.
calificacion_4 = Relación precio-calidad.
